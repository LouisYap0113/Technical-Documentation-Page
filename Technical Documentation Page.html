Introduction

Today, we are going to learn some basic CSS style properties which can be used to make your text more interesting.


These style properties include:

  • color
  • font-size
  • font-family
  • font-weight
  • text-transform
color

When we want to change the color of the text, we could use this style property: color.


1st example:

color: red

Result:

Hello World

2nd example:

color: blue

Result:

Hello World
font-size

When we want to change the size of the text, we could use this style property: font-size.


1st example:

font-size: 15px

Result:

Hello World

2nd example:

font-size: 25px

Result:

Hello World
font-family

When we want to change the font for the text, we could use this style property: font-family.


1st example:

font-family: monotype corsiva

Result:

Hello World

2nd example:

font-family: forte

Result:

Hello World
font-weight

When we want to make the text looks bold, we could use this style property: font-weight.


1st example:

font-weight: 300

Result:

Hello World

2nd example:

font-weight: 900

Result:

Hello World
text-transform

When we want to change the appearance of the text, we could use this style property: text-transform.


1st example:

text-transform: lowercase

Result:

Hello World

2nd example:

text-transform: uppercase

Result:

Hello World

3rd example:

text-transform: capitalize

Result:

Hello World
